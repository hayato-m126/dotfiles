# dotfiles

設定ファイルを管理するリポジトリ

## environment

- Ubuntu 22.04 / 24.04
- macOS Sonoma / Sequoia

## usage

1. HOMEフォルダ内に、設定ファイルを置く。
2. ./install.shで設定を反映させる。
